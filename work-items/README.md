# Work Items

This directory contains all work items for the KarimAve and Officetel Registration projects.

## Organization

Work items are organized into subdirectories:
- `active/` - Currently in-progress work items
- `completed/` - Finished work items
- `backlog/` - Planned future work items

## Creating a New Work Item

1. Copy the template from `../templates/work-item-template.md`
2. Fill in all relevant sections
3. Save the file with a descriptive name in the appropriate directory
4. Update this README with a link to the new work item
